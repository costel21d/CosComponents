
### Usage

    import RoundCapitalLetter from 'react-native-cos-round-component-capital-letter';

    <View>
      <RoundCapitalLetter
        letterColor
        text="Aasdas as da"
        size={64}
        textSize={32}
        colorsList={['pink', 'green']}
      />
    </View>

`letterCollor={true}` - use associated color for letter background
